This is a **responsive e-commerce style website** created as a front-end practice project.

## Description
IndieShop is a static shop showcase focused on layout, styling, and responsive design.  
It simulates a modern online store with a product grid, category navigation, and UI interactions.

## Features
- Fully responsive design (mobile, tablet, desktop)
- Product cards with hover effects
- Sticky header and banner section
- CSS Grid and Flexbox layout
- CSS variables and reusable utility classes
- Image fetching from Unsplash
- Font Awesome icons

## Technologies Used
- **HTML5**
- **CSS3**
- No frameworks
- No JavaScript (UI-only practice) just for fetching
